# Industrial Inventory Tracker

A simple web application to manage industrial equipment inventory. This project allows users to **view, add, edit, and delete equipment** with details such as name, type, status, and last cleaned date.

## Features

- View all equipment in a clean table format  
- Add new equipment with a form  
- Edit existing equipment details  
- Delete equipment  
- Search equipment by name  
- Sort equipment by last cleaned date  
- Responsive UI for desktop and mobile  

## Technology Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express  
- **Database:** JSON file (can be upgraded to MySQL)  
- **Styling:** CSS
